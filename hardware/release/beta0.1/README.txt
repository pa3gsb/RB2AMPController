RB2AMPController beta 0.1 Release

## Project Description

RB2AMPController (Radioberry V2.0 Amplifier Controller)

This board will be part of an amplifier. 

The ethernet connection of the board does control the amplifier.



## Folders

- pcb 
	contains the gerber pcb files and drill file.
- position
	position files top and bottom
- bom
	contains the BOM.

## Assembly

no special info.
	
## More info

The Radioberry pico (RP2040) needs a firmware.

##Note

The schema does contain a 18 pin / 8 channel IO driver; but the footprint i selected was 16 pins.
